# Contributing

Thanks for your interest in contributing. Please follow these steps to help us maintain a high-quality codebase:

1. Fork the repository and create a feature branch: `git checkout -b feat/your-feature`
2. Write tests for new features or bug fixes.
3. Format code with `black` and run `isort`.
4. Run the test suite: `pytest`.
5. Open a pull request with a clear description of changes and link any related issues.
6. Ensure CI passes before requesting a review.
